## Usage

### Clone

[https://github.com/co851002/word-counter](https://github.com/co851002/word-counter)

### Install

##### Clone this repo and cd into directory

##### Install node packages:

```
npm install
```

### Run

```
npm start
```

### Build
##### Build for production

```
npm run build
```

### View

http://localhost:3000/
